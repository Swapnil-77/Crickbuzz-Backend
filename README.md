# Cricbuzz Backend API

## 📌 Description
Backend system to manage cricket matches, teams, and players.

## ⚙️ Tech Stack
- Java
- Spring Boot
- MySQL

## 🚀 Features
- Create teams and players
- Manage matches
- Update match scores
- REST API support

## 🔗 API Endpoints
- GET /matches
- POST /matches
- GET /players
- POST /teams

## ▶️ How to Run
1. Clone repo
2. Configure MySQL
3. Run Spring Boot application
